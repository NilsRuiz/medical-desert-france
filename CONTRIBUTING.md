# Contributing

Thanks for improving Medical Desert France.

## Development

1. Create a virtual environment.
2. Install the project with `python -m pip install -e ".[dev]"`.
3. Run `ruff check .` and `pytest` before opening a pull request.

## Project Direction

Keep the ML model simple and explainable. Contributions should prioritize reproducible
MLOps, deployment clarity, observability, tests, and documentation.

Use public aggregate data only. Do not add personal health data.
